# **App Name**: Kover

## Core Features:

- Employee Onboarding Flow: Guides new employees through welcome, LGPD consent, and a personalized quiz to understand their insurance needs.
- Dynamic Quiz & Progress Tracking: Presents insurance-related questions one at a time, collecting user data through various input types, and displays a visual progress bar.
- AI-Powered Insurance Recommendation Tool: Utilizes quiz answers and other user-provided data to recommend tailored insurance packages from a predefined catalog.
- Interactive Plan Customization: Allows employees to customize selected insurance plans using toggles and sliders, with real-time updates of costs and benefit utilization.
- Real-time Financial Summary: Calculates and displays the total cost of selected insurances, tracks the remaining benefit balance, and shows any excess payment required.
- Company Dashboard Analytics: Provides an overview for companies with key metrics, such as total employees, protection coverage percentage, investment, and insurance distribution charts.
- B2B2C Role Switching: Enables users to toggle between the 'Employee' onboarding and selection flow and the 'Company' dashboard view.

## Style Guidelines:

- Primary brand color: Deep, vibrant purple (#6C3AED). Used for main buttons, progress indicators, and interactive elements to convey trust and innovation.
- Background color: Extremely light grey (#FAFAFA), providing a clean, modern, and spacious backdrop, reminiscent of financial tech apps.
- General accent color: A subtle blue-violet (#786EF7), analogous to the primary, used for less critical highlights and active states.
- Confirmation accent color: Bright green (#10B981), specifically reserved for success messages, positive affirmations, and confirmation buttons.
- Main typeface: 'Inter' (sans-serif) for all text. Chosen for its clean, modern, and highly legible characteristics, suitable for both large headlines and body text on mobile screens.
- Icons: Use 'Lucide React' for all icons, ensuring a consistent, crisp, and easily scalable visual language throughout the application.
- Mobile-first design: Optimized for a maximum width of 430px, centrally aligned. Features include soft-edged cards for content grouping, a fixed footer for crucial actions, and a prominent progress bar for guided flows.
- Transitions: Smooth horizontal slide animations between screens for a fluid user experience, particularly within multi-step processes. Confirmation states will feature a subtle checkmark animation.
